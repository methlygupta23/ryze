## ğŸš€ Deployment

This project is configured for easy deployment on Vercel.

### Deploy to Vercel

1. **Push your code to GitHub** (already done)

2. **Import your repository to Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository `methlygupta23/ryze`
   - Select the `ryze-redesign` folder as the root directory
   - Configure your build settings (Vercel usually auto-detects)
   - Add environment variables if needed
   - Click "Deploy"

3. **Automatic Deployments:**
   - Every push to your main branch will trigger a new deployment
   - Pull requests will generate preview deployments

### Alternative Deployment Methods

#### Using Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Deploy to production
vercel --prod
```
